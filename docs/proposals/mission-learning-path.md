## Objective

Transform missions from conversation-only experiences into complete learning units.

---

## Current Flow

Mission
→ Tutor Chat
→ Pronunciation Assessment
→ XP
→ Completion

---

## Proposed Flow

Mission
→ Learning Guide
→ Practice Activities
→ Tutor Chat
→ Pronunciation Assessment
→ XP
→ Completion

---

## Selected Option

Hybrid Model

Static Content:

- Objectives
- Vocabulary
- Grammar Focus
- Examples

AI Generated Content:

- Mini Quizzes
- Fill-in-the-blanks
- Translation Exercises

---

## Benefits

- Consistent pedagogy
- Low AI cost
- Better pronunciation scores
- Better grammar scores
- Better student retention

## Risks

- New database entities
- Additional frontend screens
